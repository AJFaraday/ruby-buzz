ruby-buzz
==========

A Ruby library for controlling the LEDs on buzz controllers in Linux

Scripts
=======

Turn on buzzer 1

`ruby scripts/light.rb 1 on`

Turn off buzzer 1

`ruby scripts/light.rb 1 off`
